一、操作系统中文件和目录权限的符号表示及其含义的笔记：

权限符号解释

符号 意义 对应权限
- 普通文件 无额外权限
d 目录 无额外权限（仅在目录条目上表示）
l 符号链接 无额外权限
c 字符设备文件 无额外权限
b 块设备文件 无额外权限
p 命名管道 无额外权限

用户分组和权限

  • 用户（User）：表示文件或目录的所有者。
  • 组（Group）：表示与所有者属于同一组的用户。
  • 其他（Other）：表示所有不属于用户或组的用户。

权限级别

  • 读（r）：允许用户读取文件内容或目录内容。
  • 写（w）：允许用户修改或删除文件，创建或删除目录内容。
  • 执行（x）：允许用户进入目录或执行文件（对于非目录称为可执行）。

权限符号具体说明

权限组合 用户权限 组权限 其他权限
r-- 读 无 无
-w- 写 无 无
--x 无 无 执行
r-x 读/执行 无 执行
rw- 读/写 无 无
rwx 读/写/执行 无 执行
r-- 读 读 无
… 其他组合类推

在权限表示中，没有权限的情况下通常不显示，比如 -r-x 表示只有执行权限。同时，权限的读、写、执行可以组合使用。

权限设置示例

  • rwxr-xr-x 表示所有者有读、写、执行权限，组和其他用户有读和执行权限。
  • rw-r--r-- 表示所有者有读和写权限，组和其他用户只有读权限。
  • rwx------ 表示所有者有全部权限，组和其他用户没有任何权限。
